一、设计风格总览
--------

### 1.1 设计理念

**关键词**：科技感、透明、速度、数据流

**灵感来源**：

* VS Code 深色编辑器风格

* Postman 现代版设计

* 数据可视化仪表板风格

* 科幻电影中的HUD界面元素

**核心感觉**：让用户感觉自己正在操控一个专业的技术工具，而非普通的网页应用

### 1.2 色彩体系

**主色调 - 科技蓝紫渐变**

| 色值名称  | 色码                | 用途              |
| ----- | ----------------- | --------------- |
| 深空黑背景 | #0A0E17           | 主背景色            |
| 面板底色  | #111827           | 侧边栏、卡片背景        |
| 悬浮高亮  | #1F2937           | 鼠标悬停、选中状态       |
| 科技蓝   | #3B82F6           | 主要操作按钮、链接       |
| 霓虹紫   | #8B5CF6           | 辅助操作、特殊标记       |
| 渐变蓝紫  | #3B82F6 → #8B5CF6 | Logo、重要按钮、选中指示条 |
| 成功绿   | #10B981           | GET方法、成功状态      |
| 警告橙   | #F59E0B           | POST/PUT方法      |
| 危险红   | #EF4444           | DELETE方法、错误状态   |
| 信息青   | #06B6D4           | 信息提示、中性操作       |
| 文字主色  | #F3F4F6           | 主要文字            |
| 文字辅助  | #9CA3AF           | 次要文字、占位符        |
| 边框色   | #1F2937           | 分割线、边框          |

**配色原则**：

* 深色为主，减少眼部疲劳

* 高饱和亮色仅用于强调和操作引导

* 渐变仅在Logo、主要CTA按钮、选中指示条使用，不过度使用

### 1.3 字体规范

| 用途    | 字体                                       | 字号        | 字重      |
| ----- | ---------------------------------------- | --------- | ------- |
| 界面主字体 | 'JetBrains Mono', 'Fira Code', monospace | 13px      | 400     |
| 代码内容  | 'JetBrains Mono', 'Fira Code', monospace | 12px      | 400     |
| 标题    | 'Inter', system-ui                       | 14px-20px | 500-600 |
| 按钮文字  | 'Inter', system-ui                       | 13px      | 500     |
| 小字提示  | 'Inter', system-ui                       | 11px      | 400     |

**字体加载策略**：

* 优先使用系统等宽字体

* 可选加载 JetBrains Mono（CDN方式）

* 降级方案：monospace

### 1.4 圆角与间距

| 元素类型  | 圆角大小 | 示例        |
| ----- | ---- | --------- |
| 按钮    | 6px  | 发送按钮、解析按钮 |
| 卡片/面板 | 8px  | 侧边栏、主内容区  |
| 输入框   | 6px  | URL输入、文本域 |
| 标签/徽章 | 12px | 方法标签、状态码  |
| 模态框   | 12px | 弹窗        |

**间距系统**（基于4px网格）：

* xs: 4px

* sm: 8px

* md: 12px

* lg: 16px

* xl: 24px

* 2xl: 32px

### 1.5 动效规范

| 交互      | 时长    | 缓动函数                         |
| ------- | ----- | ---------------------------- |
| 按钮点击    | 150ms | ease-out                     |
| 面板展开/收起 | 200ms | cubic-bezier(0.4, 0, 0.2, 1) |
| 页面切换    | 250ms | ease-in-out                  |
| 悬停效果    | 100ms | ease                         |
| 加载状态    | 循环    | 不适用                          |

**动效原则**：动效应为功能性服务，不炫技、不拖沓

三、交互设计细节
--------

### 3.1 科技感动效清单

| 场景   | 动效描述                        |
| ---- | --------------------------- |
| 页面加载 | 内容从下往上淡入，持续250ms            |
| 发送请求 | 发送按钮显示loading状态，响应区域出现扫描线效果 |
| 解析成功 | 解析按钮出现光波扩散效果，持续300ms        |
| 保存成功 | 保存按钮短暂显示✓图标，然后恢复            |
| 删除操作 | 条目向左滑动淡出                    |
| 面板切换 | Tab内容淡入淡出切换                 |
| 拖拽排序 | 拖拽时有半透明克隆体，目标位置显示插入指示线      |

### 3.2 光标与悬停效果

| 元素    | 光标           | 悬停效果         |
| ----- | ------------ | ------------ |
| 按钮    | pointer      | 亮度提升10%，阴影加深 |
| 输入框   | text         | 边框变为科技蓝      |
| 树节点   | pointer      | 背景变亮         |
| Tab标签 | pointer      | 文字颜色变亮       |
| 可拖拽元素 | grab/grabing | 透明度变化        |

### 3.3 加载状态

**骨架屏**：

* 首次加载时显示灰色块占位

* 带有流动的渐变光效

**发送请求中**：

* 发送按钮变为禁用状态，显示转圈图标

* 响应区域显示“等待响应...”+ 脉冲动画

* 可点击“取消”按钮中止请求

**压力测试中**：

* 显示实时进度条

* 数字计数器滚动更新

* 图表实时刷新

### 3.4 通知与反馈

**Toast通知**：

* 位置：右上角

* 持续时间：成功2秒 / 错误4秒

* 样式：深色背景，带图标

**确认对话框**：

* 毛玻璃背景

* 标题+内容+取消/确认按钮

* 危险操作为红色确认按钮

### 5.1 开发环境搭建

**前端环境**：

bash

# 1. 安装Node.js 18+

# 2. 创建Vue3项目

npm create vite@latest api-debug-tool -- --template vue
cd api-debug-tool

# 3. 安装依赖

npm install
npm install element-plus pinia axios monaco-editor
npm install @vueuse/core -D

# 4. 启动开发服务器

npm run dev

**后端环境**：

bash

# 1. 安装Python 3.10+

# 2. 创建虚拟环境

python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. 安装依赖

pip install fastapi uvicorn httpx pydantic

# 4. 创建项目结构

mkdir -p app/routers app/schemas app/services

# 5. 启动服务

uvicorn app.main:app --reload --port 8000
