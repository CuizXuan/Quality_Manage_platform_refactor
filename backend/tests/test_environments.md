# 环境管理模块测试用例

## 功能描述
环境管理模块负责管理不同测试环境及其变量，支持默认环境设置，环境间变量隔离。

## API 端点
| 方法 | 端点 | 功能 |
|------|------|------|
| GET | `/api/environments` | 获取环境列表 |
| POST | `/api/environments` | 创建环境 |
| GET | `/api/environments/{env_id}` | 获取单个环境 |
| PUT | `/api/environments/{env_id}` | 更新环境 |
| DELETE | `/api/environments/{env_id}` | 删除环境 |
| POST | `/api/environments/{env_id}/set-default` | 设置默认环境 |

## 边界条件清单
- 环境名称超长（>50字符）
- 变量 JSON 格式错误
- sort_order 为负数
- 同时设置多个默认环境（应只有一个）

## 异常场景清单
- 环境不存在（404）
- 变量格式错误（非合法JSON）
- 删除被用例/场景引用的环境
