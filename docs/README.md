# Quality Manage Platform 开发规范索引

本目录用于指导后续模型和开发者在新脚手架上继续开发。当前代码只保留登录页与系统管理底座，业务模块需要按本文档逐步补齐。

2. [核心业务链路与数据流](./02_核心业务链路与数据流.md)
3. [终端、用例、场景体系](./03_终端用例场景体系.md)
4. [AI 功能实现规范](./04_AI功能实现规范.md)
5. [后端开发规范](./05_后端开发规范.md)
6. [前端组件与页面规范](./06_前端组件与页面规范.md)
7. [开发顺序与验收标准](./07_开发顺序与验收标准.md)

## 使用方式

- 开发新模块前，先阅读 `01` 和对应模块说明。
- 打通跨模块链路前，先阅读 `02`。
- 开发接口调试、用例、场景前，必须阅读 `03`。
- 开发 AI 能力前，必须阅读 `04`。
- 写后端代码前，参考 `05`。
- 写前端页面和组件前，参考 `06`。
- 拆任务和验收时，参考 `07`。
