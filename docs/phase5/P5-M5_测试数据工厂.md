# P5-M5: 测试数据工厂

## 1. 模块概述

**功能**: 统一管理测试数据的生成、脱敏、克隆和回收

**核心能力**:
- 数据脱敏规则引擎
- 测试数据生成器
- 环境间数据克隆
- 数据快照管理

## 2. 数据库设计

### 2.1 数据脱敏规则表 `data_mask_rule`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTO | 自增主键 |
| name | VARCHAR(200) | NOT NULL | 规则名称 |
| field_pattern | VARCHAR(500) | NOT NULL | 字段匹配模式(正则/JSONPath) |
| mask_type | VARCHAR(50) | NOT NULL | 脱敏类型 |
| mask_config | TEXT | | 脱敏配置 JSON |
| enabled | BOOLEAN | DEFAULT 1 | 是否启用 |
| priority | INTEGER | DEFAULT 0 | 优先级 |
| project_id | INTEGER | FK | 所属项目 |
| created_at | TIMESTAMP | DEFAULT | 创建时间 |

**脱敏类型**:
| 类型 | 说明 | 示例 |
|------|------|------|
| phone | 手机号脱敏 | 138****1234 |
| email | 邮箱脱敏 | t***@example.com |
| id_card | 身份证脱敏 | 110***********1234 |
| bank_card | 银行卡脱敏 | 6222********1234 |
| password | 密码脱敏 | ****** |
| token | Token 脱敏 | eyJ***... |
| custom | 自定义正则 | 按配置替换 |

## 3. API 设计

### 3.1 脱敏规则

```
GET /api/test-data/mask-rules?project_id=1
POST /api/test-data/mask-rules
Request:
{
  "name": "手机号脱敏",
  "field_pattern": "$.phone|$.mobile",
  "mask_type": "phone"
}

PUT /api/test-data/mask-rules/{id}
DELETE /api/test-data/mask-rules/{id}
```

### 3.2 数据生成

```
POST /api/test-data/generate
Request:
{
  "template_id": 1,
  "count": 100,
  "unique_fields": ["email", "phone"]
}

Response:
{
  "data": [
    {"id": 1, "name": "张三", "email": "zhangsan@example.com", "phone": "138****1234"},
    ...
  ]
}
```

### 3.3 数据克隆

```
POST /api/test-data/clone
Request:
{
  "name": "生产数据克隆到测试环境",
  "source_env_id": 1,
  "target_env_id": 2,
  "tables": ["users", "orders"],
  "mask_rules": [1, 2, 3],
  "clone_type": "mask"
}
```

### 3.4 数据快照

```

```

## 4. 服务层设计

### 4.1 `DataMaskService.py`

```python
class DataMaskService:
    """数据脱敏服务"""
    
    MASK_TYPES = {
        'phone': lambda v, cfg: self._mask_phone(v),
        'email': lambda v, cfg: self._mask_email(v),
        'id_card': lambda v, cfg: self._mask_id_card(v),
        'bank_card': lambda v, cfg: self._mask_bank_card(v),
        'password': lambda v, cfg: '******',
        'token': lambda v, cfg: self._mask_token(v),
        'custom': lambda v, cfg: self._mask_custom(v, cfg)
    }
    
    def apply_mask(self, data: dict, rules: list) -> dict:
        """应用脱敏规则"""
        
    def preview_mask(self, data: dict, rule_id: int) -> dict:
        """预览脱敏结果"""
```

class DataGenService:
