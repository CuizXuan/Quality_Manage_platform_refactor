# P5-M4: 混沌工程引擎

## 1. 模块概述

**功能**: 通过主动故障注入验证系统韧性

**故障类型**:
| 类型 | 配置项 | 说明 |
|------|--------|------|
| cpu-stress | cpu_percent, duration | CPU 压力 |
| memory-stress | memory_percent, duration | 内存压力 |
| network-delay | delay_ms, jitter_ms, duration | 网络延迟 |
| network-loss | loss_percent, duration | 网络丢包 |
| pod-kill | pod_name, force | 杀死 Pod |
| service-down | service_name, duration | 服务下线 |
| disk-fill | path, size_gb, duration | 磁盘填充 |
| dns-failure | domain, duration | DNS 故障 |

## 3. API 设计

### 3.1 混沌实验

```
POST /api/chaos/experiments
Request:
{
  "name": "订单服务韧性测试",
  "description": "验证订单服务在支付服务延迟时的表现",
  "target_type": "service",
  "target_id": "order-service",
  "hypothesis": "即使支付服务延迟增加 2s，订单服务仍能正常响应",
  "steady_state": {
    "tps_min": 100,
    "error_rate_max": 0.01,
    "p95_latency_ms": 500
  },
  "blast_radius": 30,
  "auto_rollback": true,
  "rollback_condition": {"error_rate": 0.1}
}
```

```

    "delay_ms": 2000,

```

```python
    """混沌实验服务"""

    def start_experiment(self, experiment_id: int) -> dict:

    def pause_experiment(self, experiment_id: int):
        """暂停实验"""
        
    def stop_experiment(self, experiment_id: int):
        """停止实验"""
        
    def get_experiment_status(self, experiment_id: int) -> dict:
        """获取实验状态"""
```

### 4.2 `FaultInjectionService.py`

```python
class FaultInjectionService:
    """故障注入服务"""
    
    FAULT_TYPES = {
        'cpu-stress': CpuStressFault,
        'memory-stress': MemoryStressFault,
        'network-delay': NetworkDelayFault,
        'network-loss': NetworkLossFault,
        'pod-kill': PodKillFault,
        'service-down': ServiceDownFault,
        'disk-fill': DiskFillFault,
        'dns-failure': DnsFailureFault
    }
    
    def inject_fault(self, experiment_id: int, fault_type: str, config: dict) -> dict:
        """注入故障"""
        
    def rollback_fault(self, fault_id: int):
        """回滚故障"""
        
    def verify_steady_state(self, experiment_id: int) -> dict:
        """验证稳态"""
```
